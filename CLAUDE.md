## DESIGN SYSTEM
Design system is entirely based on Shad CN. 
When you create pages, always use Shad CN components. 

## DATABASE
Use Convex as our database.

## APP
The app uses Next.js.

## DEPLOYMENT
For deployment use Vercel.

## TEST
Use agent-browser cli to test the app.
Can also write tests.
Suggest also github actions for CI/CD if you think it's necessary.

## LINT/FORMAT FIX
If you modified at least one file, always run `pnpm fix`
